Googleスプレッドシートにデータ出力プラグインの設定

スプレッドシートID（必須）
https://docs.google.com/spreadsheets/d/xxxxxxxxxxx/edit の xxxxxxxxxxx の部分
client email（必須）
Google Developer Console で作成したサービスアカウントの client_email
このメールアドレスを上のスプレッドシートに編集権限で招待することも忘れずに
private key（必須）
Google Developer Console で作成したサービスアカウントの private_key
「新しい鍵を作成」で作成したJSONファイル内に書かれている、とても長い文字列
シート名
スプレッドシートに作成・更新するシートの名前を設定できます（デフォルトは「kintone連携」）
出力する列の指定
フィールドコードをカンマ（,）区切りで入力することで、出力する列の一覧と順番を指定できます
ボタン名
ボタンに表示する文字列を変更できます（デフォルトは「スプシに出力」）
